# AryaPars - Corporate Website
## Project images
![alt text](https://github.com/mehrdadsadeghizadeh/Corporate-website/blob/master/public/images/aboutus/Beige%20New%20Product%20Mockup%20Now%20Available%20Facebook2.png)

**AryaPars** is a corporate portfolio website that allows users to log in, browse products by categories, and place orders. Built with React.js, the site is fully responsive and provides a seamless user experience across devices.

View the live demo: [AryaPars Live Demo](https://aryapars.liara.run/)

## Features
- **User Authentication:** Users can sign up and log in to their accounts.
- **Product Categories:** View products categorized for easy browsing.
- **Order Placement:** Add products to the cart and place orders.
- **Responsive Design:** Fully responsive layout for mobile and desktop devices.
- **Toast Notifications:** Alerts and success messages using `react-toastify`.
- **Image Sliders:** Interactive product sliders powered by `swiper`.
- **Icons:** Stylish icons with `react-icons` for enhanced UI.
- **Animations:** Smooth animations on scroll using `react-animate-on-scroll`.

   ```bash

5. Start the development server:
   ```bash

## Technologies & Tools Used
- **React.js:** JavaScript library for building interactive user interfaces
- **react-toastify:** For displaying notifications and alerts
- **swiper:** For creating interactive and beautiful sliders
- **react-icons:** For adding modern and scalable icons to the UI
- **react-animate-on-scroll:** For adding animations on scroll to enhance user experience
- **CSS3:** For styling and responsive design
